Installation
-------------------------------------------------------------------------------

Clone the repo

```sh
git clone https://github.com/linssen/better_than_predator
```

Install the requirements packages

```sh
yarn install
```

Build the static files

```sh
# Build the production ready files
yarn run build
# Watch the files to automatically build them on change
yarn run serve
# Test
yarn run test
```
